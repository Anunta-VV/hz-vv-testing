# Documents Folder

This folder contains all documentation related to Horizon VV (Validation & Verification) testing.

## Structure

- status-reports/
  Contains testing status reports (FC, HCF,RC, GA, etc.)

- config-guides/
  Contains configuration and setup guides

- market-ready/
  Contains market readiness and release documents

- peripheral-testing/
  Contains device validation reports (Zebra, ThinOS, printers, etc.)

## Versioning

Each document is organized by Horizon version (e.g., 8.18, 8.17).

Example:
documents/status-reports/8.18/

## Naming Convention

- Status Report: status-<version>-<phase>-v<docversion>.xlsx  
- Config Guide: config-guide-<version>-v<docversion>.docx  
- Market Ready: mr-<version>-v<docversion>.pptx  
- Peripheral: peripheral-<device>-<version>-v<docversion>.xlsx  

## Notes

- Do not overwrite files — always create new versions (v1, v2, v3)
- Maintain proper naming for easy tracking
